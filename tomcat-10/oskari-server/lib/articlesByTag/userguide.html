This user guide contains information about the map window and its functions. In the map window you can compile map views and browse geographic datasets by opening map layers one at a time or several transparent levels on top of one another. You can also embed maps on your own website or share a link with external users. This guide contains information about how the map window works. In addition, the map window offers interactive guidance in various situations.

How to use the map

A map consists of different map layers, which have been made available by the service provider. The first thing you see when you open the map window is the default view. You can delete your map whenever you wish by clicking Restore defaults in Tools at the bottom left and in the centre of the panning controls.

 

Basic functions

Move the map by clicking and dragging it or by clicking the panning controls in the top right corner. When you are looking for a specific object or location, use the Search function.

Zoom in and out by clicking the + and - signs next to the scale selector, or click and drag the switch to directly select a specific scale level.

The coordinates show the location of the mouse pointer on the map.

The Index map shows the map view in a smaller scale. You can show and hide the index map by clicking the right bottom corner of the map window.

Objects on the map often contain additional information. Click an object on the map to view any additional information. Data products display on a list any objects that are located within the area shown in the map window. You can set conditions to limit your selection to particular kinds of objects in a data product. A list of object information is shown when you click the Object data link to the right of the map.

 

Menus

Use the Search menu to find a place name and centre the map on that location.

Use the Map layers menu to search for and select datasets provided by various providers. The datasets have been sorted according to theme and data provider. Datasets:

A grey symbol means that the dataset concerned cannot be viewed on the selected scale level. When you select a data product, an Object data link is shown to the right of the map. Click this link to view a list of object data about objects in the area covered by the map.

Use the Selected map layers menu to see which datasets are currently displayed in the map view. You can adjust the order and transparency of map layers or delete them. A bubble to the right of the menu shows the number of selected map layers.

Use the My information menu to see all saved My objects, My views (see Tools below) and the maps you have embedded elsewhere. Here you can also "un-embed" a map. The menu also contains your user information and allows you to change it. You must be logged in to use this function.

Use the Create map menu to create the code you need to embed the map view you have compiled, which you then can embed on your own website or on a company website. You need to be logged in to use this function.

The Map legend menu contains a description of the symbols used on the map layers.

The User guide menu contains this user guide. If you need more help, the i-symbol or more information link contain additional information.
Please note, that you can always reset the default settings and undo or redo any changes. These tools are located in the left margin below the menus (see Tools).

Map view

Create your own map view by selecting map layers in the Map layers menu or from the map layers you have saved in My map layers. Use the Selected map layers menu to change the order of your map layers by clicking and dragging and to change the transparency of your map layers.

How to compile a map view:
1.    Select the map layers you want to display in the Map layers menu
2.    Use the Selected map layers menu to change the order of your map layers by clicking and drag-ging them in the map layer stack, and to change their transparency if necessary

How to save a map view
Use the Save tool to save a map view. Information about selected map layers, their order and transparency, selected location and scale level will be saved in My map views. Saved map views are listed on the My map views tab in the My data menu.

How to share a map view
The Link to map view tool creates a direct link to a compiled map view. You can use it to share the map view with other Internet users. Please note that you cannot yet share your own places and map layers.

How to create and embed a map view

You can compile a map view and make it public by embedding it on another website by copying and pasting a few lines of html-code. Start by clicking the Create map menu. You need to be logged in to use the feature. Log in or Register (linkki).

To create a map, you simply give the map a name, define its size and type the URL of the website where the map will be embedded. You can also select the tools that will be shown on the map, such as the scale selector and place name search function.

Finally, you will be given a few lines of html-code. Copy and paste it in to the webpage where you want to display the map.

 

My places/objects

You can add your own places/objects to the map. An object can be a dot, a route or an area with a name and a description. When you have marked an object on the map, click "Save" to save it on one of your own map layers or create a new map layer. You can use different map layers to look at objects belonging to the same theme together. You need to be logged in to save, edit and delete your own objects.

Where are My places?
The objects/places you save can be found in the My data menu on the Places tab.

How to view My places:
Click the name of an object on the Places tab to view it.

How to share My places:
Currently, it is not possible to share links to objects/places that you have saved.

How to edit My places:
Use the Edit function to edit the name, description or map layer of a saved place/object.

How to delete My places:
Use the Delete function to delete a saved place/object from the map layer.

 

My map layers

You can save objects/places on your map layers. Use different map layers to separate one theme from another. You can find them by selecting My map layers from the drop-down menu on the Places tab on the My data menu.

How to create a new map layer:
When you save or edit a place/object, you automatically create a map layer. (You must be logged in to save or edit.)

How to view My map layers:
You can view one of the map layers you have saved when you click an object in the My places list or by ticking the layer My places in the category Places in the Map layers menu.

How to edit and delete My map layers:
You can edit the name or style of a map layer by clicking the Edit option.

The privacy settings of My map layers:
My map layers and the objects/places they contain are private by default.

How to make My map layers public:
Currently, it is not possible to make user-created map layers public.
